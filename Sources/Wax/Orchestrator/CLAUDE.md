# Sources/Wax/Orchestrator Module

## Purpose
Contains extensions to `MemoryOrchestrator` for domain-specific ingestion pipelines. Currently hosts the PDF ingestion extension.

## Key Types

| Type | Role |
|------|------|
| `MemoryOrchestrator+PDF` | Extension adding `remember(pdfAt:metadata:)`. Extracts text from a PDF and ingests it with source metadata. |

## Architecture

- `MemoryOrchestrator` itself lives elsewhere in the Wax target (general-purpose text memory orchestrator).
- This directory holds only extensions that add new ingestion modalities.
- PDF ingestion: extracts text via `PDFTextExtractor`, merges host-supplied metadata with PDF-specific keys, then delegates to the orchestrator's core `remember(_:metadata:)` method.

## Metadata Convention

PDF ingestion writes these metadata keys:
- `source_kind` = `"pdf"`
- `source_uri` = absolute URL string
- `source_filename` = last path component
- `pdf_page_count` = string-encoded page count

## Dependencies

- `PDFTextExtractor` (from Ingest module)
- `PDFIngestError` (from Ingest module)
- `Foundation`
